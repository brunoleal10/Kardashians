Tecnologias

A equipe usa várias ferramentas, conforme as exigências do problema a ser resolvido. De forma geral, usamos PostgreSQL como Sistema Gerenciador de Banco de Dados; XAMPP para usar apache; Netbeans ou Lazarus como editor de códigos; e Git para o controle de versões.
